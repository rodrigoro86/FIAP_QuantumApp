QuantumApp

Uma Solução Inovadora da QuantumFinance

Sobre o Projeto

A QuantumFinance contratou a consultoria SmartMind Analytics para desenvolver o QuantumApp, um aplicativo projetado para ser simples e fácil de operar, atendendo às necessidades dos nossos clientes de maneira eficiente e eficaz.

Requisitos do Aplicativo

  • Cadastro de Cliente: Inclusão de fotos, assinatura eletrônica e informações de contato.
  • Processo de Transações: Suporte para DOC, TED e Pix.
  • Recomendação de Produtos: Sugestões de produtos de crédito na página inicial.
  • Análise de Transações: Telas e relatórios para análise detalhada das transações da conta (por período e por tipo de movimento).

Entregáveis

O projeto inclui os seguintes entregáveis:

  • Diagramas de modelagem de dados
  • Scripts de criação dos bancos de dados gerados automaticamente pelas ferramentas de modelagem
